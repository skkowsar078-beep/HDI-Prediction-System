About HDI Prediction System

The Human Development Index (HDI) Prediction System is a Machine Learning web application that predicts the HDI category of a country based on important socio-economic indicators.

Project Features

  • 🌍 Predict HDI Category
  • 📊 Machine Learning Model (Random Forest)
  • 📈 Accuracy: 90.47%
  • 📜 Prediction History
  • 🔐 Secure Login System
  • 💻 Interactive Dashboard

Dataset

Official Human Development Report (UNDP)

Input Features

  • Life Expectancy
  • Expected Years of Schooling
  • Mean Years of Schooling
  • Gross National Income (GNI) Per Capita

Output

HDI Categories:

  • 🟢 Very High
  • 🔵 High
  • 🟡 Medium
  • 🔴 Low
